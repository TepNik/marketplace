### SignatureInfo

```solidity
struct SignatureInfo {
  address user;
  bool isTokenToGetMulti;
  bool isTokenToGiveMulti;
  struct EIP712.TokenInfo tokenToGet;
  struct EIP712.TokenInfo tokenToGive;
  uint256 deadline;
}
```
